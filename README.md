# 💖 Aşk Dolu Kalp

Sevgililer Günü'ne özel, romantik bir web sayfası.  
Kalp yağmuru efekti, özel mesajlar ve sürpriz butonu ile sevdiğinize tatlı bir jest yapın.

## ✨ Özellikler

- 🌧️ Sürekli kalp yağmuru animasyonu  
- 💌 Duygusal aşk mesajı  
- 🎉 "Sana Özel Bir Sürpriz" butonu (rastgele romantik mesajlar)  
- 🖱️ Karta tıklayınca minik kalp patlamaları  
- 📱 Tamamen duyarlı (responsive) tasarım  
- 🎨 Cam efektli (glassmorphism) modern görünüm
